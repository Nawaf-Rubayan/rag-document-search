# RAG HTML Tool (Simple Retrieval-Augmented Generation)

A lightweight browser-based RAG (Retrieval-Augmented Generation) tool built with a single HTML file.  
It allows users to upload documents, create an index, and perform semantic search using an LLM API.

---

## 🚀 Features

- Simple **single HTML file** (no backend required)
- Add your own **API Key**
- Choose between different **LLM models**
- Upload documents directly from your device
- Automatically **index documents**
- Perform **semantic search over your files**
- Lightweight and easy to use

---

## 📸 How It Works

1. Open the HTML file in your browser
2. Enter your API key
3. Select the model you want to use
4. Upload your file(s)
5. Click **Index**
6. Start asking questions and search your data

---

## 🧠 Use Case

This tool is useful for:
- Quick document search using AI
- Testing RAG pipelines
- Learning how embeddings + retrieval work
- Lightweight personal knowledge base

---

## ⚙️ Setup

No installation required.
